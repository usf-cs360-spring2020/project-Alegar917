The datasets that I used was Data SF Eviction Notices and American Community Survey San Francisco Neighborhoods Socio-Economic Profiles 2012-2016

Data SF dataset shows the Eviction Notices filed with the San Francisco Rent Board per San Francisco Administrative Code 37.9(c).

Notices are published since January 1, 1997 and monthly updated (as of this date in April 2020).Data was accessed in April 13, 2020

All data are under Public Domain Dedication and License (PDDL).

The data itself is divided into 29 columns and 41k rows. These are the columns I used:

Field Name Data Type Definition
File Date Date & Time The date on which the eviction notice was filed with the Rent Board of Arbitration.
Non Payment Checkbox This field is checked (true) if the landlord indicated non-payment of rent as a grounds for eviction.
Breach Checkbox This field is checked (true) if the landlord indicated breach of lease as a grounds for eviction.
Nuisance Number Checkbox This field is checked (true) if the landlord indicated nuisance as a grounds for eviction.
Illegal Use Checkbox This field is checked (true) if the landlord indicated an illegal use of the rental unit as a grounds for eviction.
Failure to Sign Renewal Checkbox This field is checked (true) if the landlord indicated failure to sign lease renewal as a grounds for eviction.
Access Denial Checkbox This field is checked (true) if the landlord indicated unlawful denial of access to unit as a grounds for eviction.
Unapproved Subtenant Checkbox This field is checked (true) if the landlord indicated the tenant had an unapproved subtenant as a grounds for eviction.
Owner Move In Checkbox This field is checked (true) if the landlord indicated an owner move in as a grounds for eviction.
Demolition Checkbox This field is checked (true) if the landlord indicated demolition of property as a grounds for eviction.
Capital Improvement Checkbox This field is checked (true) if the landlord indicated a capital improvement as a grounds for eviction.
Substantial Rehab Checkbox This field is checked (true) if the landlord indicated substantial rehabilitation as a grounds for eviction.
Ellis Act WithDrawal On Scene DtTm Checkbox This field is checked (true) if the landlord indicated an Ellis Act withdrawal (going out of business) as a grounds for eviction. coordinates.
Condo Conversion Checkbox This field is checked (true) if the landlord indicated a condo conversion as a grounds for eviction.
Roommate Same Unit Checkbox This field is checked (true) if the landlord indicated if they were evicting a roommate in their unit as a grounds for eviction.
Other Cause Disposition Checkbox This field is checked (true) if some other cause not covered by the admin code was indicated by the landlord. These are not enforceable grounds, but are indicated here for record keeping.
Late Payments Checkbox This field is checked (true) if the landlord indicated habitual late payment of rent as a grounds for eviction.
Lead Remediation Checkbox This field is checked (true) if the landlord indicated lead remediation as a grounds for eviction.
Development Checkbox This field is checked (true) if the landlord indicated a development agreement as a grounds for eviction.
Good Samaritan Ends Checkbox This field is checked (true) if the landlord indicated the period of good samaritan laws coming to an end as a grounds for eviction.

American Community Survey dataset shows the 2016 Socio-Economic Profiles for the SF neighborhoods.

This report compiles census-tract-level 2014 Five Year American Community Survey census data for each neighborhood. It provides select demographic and housing characteristics as well as information on employment and the commute to work.

Processing/Filtering

For the collected data, null values were filtered out of the visualizations.

Data Wrangling

Since the focus was to see 2019 eviction notices, only the data of file dates from 1/1/2019 to 12/31/2019 were used. Much of my wrangling was counting the number of types of eviction notices and creating a hierarchy between the type of eviction and neighborhoods. I wrangled also some data from 2016 Socio-Economic Profiles for the SF neighborhoods dataset to be used
